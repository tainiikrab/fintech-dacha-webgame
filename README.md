# Финтех-дача
<img width="245" height="226" alt="обложка" src="https://github.com/user-attachments/assets/c4ee9260-fc7b-456b-a892-ea1f8fe27c3a" />

Игра для хакатона‑геймджема «Лидеры цифровой трансформации 2025».

Проект реализован на Unity за пару дней. Игрок развивает свой дачный участок, одновременно взаимодействуя с финансовыми инструментами банка.

## Описание

«Финтех‑дача» сочетает в себе простую игровую механику роста дачи с элементами финансового управления: магазин, улучшения, мини‑игры с финансовыми продуктами (вклад, кредит, инвестиции). Основной фокус был на прототипе и интерактивности.
